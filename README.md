# Hops

### Overview
Hops is an app to view , save and compare hops.

### Technologies used
- Flutter (Dart)
- Provider Package
- Json Fetching Locally

### Functionality
- View each Hop Details.
- Dynamic Comparaison between Hops.
- Save and search hops to stay organized.
